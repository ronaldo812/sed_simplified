# Отчет по практической работе №2: Упрощенный sed
 
**Вариант:** Реализация на 15 баллов 

## Описание проекта
Программа представляет собой упрощенную версию потокового редактора `sed`, написанную на языке C. Утилита принимает имя файла и команду редактирования, изменяя содержимое исходного файла.

## Реализованный функционал
1.  **Поддержка регулярных выражений:**
    *   Использована библиотека `<regex.h>` (стандарт POSIX).
    *   Поддерживаются расширенные регулярные выражения (`REG_EXTENDED`).
2.  **Команды редактирования:**
    *   `s/regex/replacement/` — замена всех вхождений шаблона на указанный текст.
    *   `/regex/d` — удаление строк, соответствующих шаблону.
3.  **Работа с файлами:**
    *   Редактирование происходит "in-place" (изменяется переданный файл).
    *   Реализовано через создание временного файла `.tmp` и последующую замену оригинала.

## Структура проекта
Проект декомпозирован на модули согласно требованиям:
*   **`main.c`**: Точка входа. Отвечает за валидацию аргументов командной строки.
*   **`operations.h`**: Заголовочный файл с описанием функций.
*   **`operations.c`**: Основная логика:
    *   Парсинг строки команды (определение типа `s` или `d`).
    *   Компиляция регулярного выражения (`regcomp`).
    *   Построчное чтение и обработка файла.
    *   Функция `replace_all` — ручная реализация замены всех вхождений в строке с использованием `regexec` и арифметики указателей.
*   **`Makefile`**: Скрипт сборки с целями `all`, `clean`, `test`.
